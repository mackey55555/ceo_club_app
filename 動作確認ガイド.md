```bash
cd mobile-app
npm start
```

## トラブルシューティング

### Supabase接続エラーが発生する場合

1. 環境変数が正しく設定されているか確認
   ```bash
   # モバイルアプリ
   cat mobile-app/.env
   
   # 管理画面
   cat admin/.env.local
   
   # Web
   cat web/.env.local
   ```

2. Supabaseダッシュボードで接続確認
   - Settings > API でURLとキーを確認
   - 提供されたキーが `sb_publishable_` で始まる場合、新しい形式の可能性があります
   - 通常の形式は `eyJ...` で始まるJWTトークンです

### モバイルアプリが起動しない場合

1. Expo CLIがインストールされているか確認
   ```bash
   npx expo --version
   ```

2. 依存関係を再インストール
   ```bash
   cd mobile-app
   rm -rf node_modules
   npm install
   ```

### Next.jsアプリが起動しない場合

1. ポートが使用中でないか確認
   ```bash
   lsof -i :3000
   lsof -i :3001
   ```

2. 依存関係を再インストール
   ```bash
   cd admin  # または web
   rm -rf node_modules .next
   npm install
   ```
